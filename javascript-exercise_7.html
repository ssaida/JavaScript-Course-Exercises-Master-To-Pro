ג'אווה סקריפט המדריך המלא 2023
תנאים תרגיל מספר 7 ניהול תקציב

בתרגיל ג'אווה סקריפט הבא נשאל את המשתמש בכמה ביטקוין הוא מחזיק, אם הוא יכניס מספר חיובי נשאל אותו כמה ביטקוין הוא רוצה למשוך, לבסוף נדפיס את יתרת חשבונו.

  • a. prompt the user and ask him how many bitcoins he has saved as btcAmount.
  • b. if the user enters a positive number, ask him how many bitcoins he would like to withdraw now.
  • c. print the final balance after the user withdraw amount.
  • d. Bonus: if the user asks to withdraw more than his total balance, alert him that he has "insufficient balance". 

var btcAmount = +prompt('How many bitcoins do you have?')
if (btcAmount) {
    var withdraw = +prompt('How many bitcoins you like to withdraw?')
    if (btcAmount < withdraw) {
        alert('insufficient balance')
    } else {
        btcAmount -= withdraw
    }
}
console.log('Final bitcoin balance is: ', btcAmount)
