ג'אווה סקריפט המדריך המלא 2023
תנאים תרגיל מספר 6 ותק נהיגה

בתרגיל ג'אווה סקריפט הבא נלמד איך להשתמש בתנאים על מנת לקבוע ותק נהיגה של המשתמש, תחילה נקלוט מספר שישמש שנות נהיגה ואז באמצעות תנאי נבדוק את הותק.

  • a. prompt the user and ask him how many years he's been driving.
  • b. if he's been driving less than 3 years, print, "You're a young driver!".
  • c. if he's driving 3 years but less than 6 years print"You shape some sidewalks don't you?".
  • d. if the user is driving 6 years or more, print "can you be my private driver please?". 

var drivingExp = +prompt('How many years you are driving?')

if (drivingExp < 3) {
    console.log(`You're a young driver!`)
} else if (drivingExp >= 3 && drivingExp < 6) {
    console.log('You shape some sidewalks dont you?')
} else {
    console.log('can you be my private driver please?')
}
