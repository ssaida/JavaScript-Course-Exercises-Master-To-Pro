ג'אווה סקריפט המדריך המלא 2023
פונקציות תרגיל מספר 14 קריאה חוזרת callback

בתרגיל ג'אווה סקריפט הבא נלמד איך להשתמש בפונקציות בצורה מורכבת, נלמד שפונקציות יכולות לקבל בתור פרמטר גם פונקציות אחרות וכי בג'אווה סקריפט פונקציות מקבלות יחס של אזרח ראשון.

  • a. prompt the user to enter some number saved as userNumber.
  • b. write a function myFunc() that expects 2 parameters, a number and another function, use this function with the number that have been given and print the results.
  • c. write a function myCallback() that expect a number multiply it by 2 and return the result.
  • d. call myFunc() with the userNumber and myCallback(). 

const userNumber = +prompt('please enter some number')

function myFunc(num, callback) {
    const result = callback(num)
    console.log('result from the cb func is', result)
}

function myCallback(num) {
    return num * 2
}

myFunc(userNumber, myCallback)
