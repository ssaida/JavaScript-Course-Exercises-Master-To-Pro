ג'אווה סקריפט המדריך המלא 2023
לולאות תרגיל מספר 26 סכימת מאורעים

בתרגיל ג'אווה סקריפט הבא נשתמש ביכולות של לולאה ומחרוזת כדי לדעת, איך לסכום כמה פעמים מופיע כל אות במשפט שאותו שולח אלינו המשתמש.

  • a. prompt the user to enter a random sentence.
  • b. loop through this string and in the end print how many times the characters a,b,c appeared.
  • c. "apple banana canvas" will result with a = 6 b = 1 c = 1.
  • d. Bonus: count all the characters which are not abc.
  • e. alert the final result with: alert(`a:${a} times, b:${b} times, c:${c} times, unknown:${unknown} times.`) 

const init = () => {
    const userInput = prompt("please enter any sentence");
    let a = 0;
    let b = 0;
    let c = 0;
    let unknown = 0;
    for (let i = 0; i < userInput.length; i++) {
        switch (userInput[i]) {
            case "a":
                a++;
                break;
            case "b":
                b++;
                break;
            case "c":
                c++;
                break;
            default:
                unknown++;
        }
    }
    alert(
        `a:${a} times, b:${b} times, c:${c} times, unknown:${unknown} times.`
    );
};

init();
