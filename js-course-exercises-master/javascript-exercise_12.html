פונקציות תרגיל מספר 12 יצירת מספר רנדומלי

בתרגיל ג'אווה סקריפט הבא נלמד איך ליצור מעין משחק רולטה פשוט, בו המשתמש נותן לנו ערך מספרי התוכנה מגרילה מספר ובודקת האם המספר של המשתמש והמספר שהוגרל שווים

  • a. write a function getLuckyNumber().
  • b. inside it, declare an inner variable to hold your lucky number named luckyNum.
  • c. prompt the user to guess a number save it as userNumber, If the userNumber and the luckyNumber are equal. alert "wow you win the game!" else "maybe next time!".
  • d. Bonus: make sure you're not using any global variables in this exercise. 

function getLuckyNumber() {
    var luckyNumber = 3
    var userNumber = +prompt('What is your guess?')
    if (userNumber === luckyNumber) {
        console.log('Wow you win the game!')
    } else {
        console.log('Maybe next time!')
    }
}
